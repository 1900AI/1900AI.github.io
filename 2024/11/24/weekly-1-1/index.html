我步入丛林， 因为我希望生活得有意义。 我希望活得深刻 吸取生命中所有的精华 把非生命的一切都击溃 以免当我生命终结时 发现自己从来没有活过。

一场哲学课上的生日🎂

周一是我一周课最多的一天，从早八的马原到晚八的西哲一天的满课。很巧的是，有一个一起上西哲的数学系的朋友 D 也是满课，更巧的是，这一天还是他的生日，于是我见证了一场在哲学课上度过的生日。🎂

这节西哲讲的内容是“生存的困惑”，从笛卡尔提出“我思故我在”后，“我”的理性成为了人的根本，成了科技发展社会进步的准则，人类基于理性的原则生存发展了四百年，但看着当下的世界，科技的高度发达好像没有带来高度的幸福，理性指导的生活好像把人变成了工具，于是我们再次听到尼采的声音，他高呼“生命是人的根本”，对生命力的推崇是他的哲学的基调。听到这里，回头看看我平时的生活，大部分时间都在座位上对着电脑学习，这诚然很好，我也十分享受求知的快乐，但似乎，不知从什么时候开始，我的生命力在慢慢地减少……🌱

当课程进展到这一部分的内容时，陈老师说这节课先上到这里，今天是 D 同学的生日，他带了蛋糕来和大家一起庆祝生日，在 D 同学的生日致辞里，他提到“既加冠,益慕圣贤之道”，我突然理解为何他选择在哲学课上庆祝生日。在和他交谈的过程中了解到了一位特意来庆祝他的生日的同学，中学毕业之后独自环游中国，参与过职业电竞的训练，然后以社会身份在复旦学习哲学，再来到大连理工学习哲学，在和我一样的年纪，却有了丰富得多的生命体验，这好像是尼采说的那种酒神式的生活。不过幸好，幸好我也在学习哲学（虽然平时很忙，读书的速度像🐌）

又想起南京大学王恒的一段视频标题“哲学的目的，是恢复你的生命力”，感谢哲学，感谢 D 同学的生日，让我这周的生活，多了一些生命力！

世界哲学日与海上钢琴师

在世界哲学日去听了一场陈老师的讲座“思考是一种道德责任——阿伦特的哲学”，其实之前从没了解过阿伦特，只是听说陈老师是研究阿伦特的专家，带着好奇去听了听。

简要来说，阿伦特提出了根本恶和恶之平庸两个概念，“根本恶”即使人之为人变得多余的恶，这种恶在总体主义（或极权主义）和资本主义等影响下尤为突出。而“恶之平庸”是指缺乏思考的状态，在生活中，我们常常为了某些目的穷尽手段，却不曾思考目的本身，这就是恶之平庸的一种体现。💡

比起讲座内容，陈老师的经历和哲学实践更给我启发。他选择研究阿伦特的哲学，是选择了一种探讨人之为人的哲学，而不仅仅是一个学术课题。数十年来，他一直秉持着自己的哲学思想生活，包括主动开设讲座分享思想，在未来技术学院教授理想国和笛卡尔的课程……🤔

陈老师提到，在博士录取时，他反复观看电影《海上钢琴师》🎥，认为自己身上有一种1900的气质，并被这种人生的意义所打动。巧合的是，姚同学早先分享了《海上钢琴师》重映的消息，而我前两天刚好去再次看了一遍，奇妙的缘分啊。

做不完的机器学习实验

虽然前面两部分都与哲学有关，但这周大部分的时间都在做机器学习实验！！！

有五个实验内容，前四个和往年一样，难度不大但是很繁琐，实验过程belike：看一看PPT里的任务要求，构思一下实验方案，chat with GPT看看能否实现初步的效果，debug，想一个更好的可视化的方案，写报告……一个实验做下来，简单的两三个小时，复杂一点就三五小时，但真正收获的可能是最开始想方案以及后面调整方案的内容，中间基本都靠AI了，感觉虚度了不少光阴。😅

比起前四个实验，第五个实验倒是有点新，我还没做完，也许留到下周再讲。

同样很新的是一张poster和一个project，poster就是选一篇或多篇论文做一张海报，我选了一篇挺有意思的论文，可能花了十多个小时精读，然后又花了几个小时做海报，做完之后感慨：什么时候做一篇自己的poster啊。

给博客引入评论系统

由于使用 GitHub 等相关方式对用户有一定的要求，即需要有 GitHub 账号，这对我的部分朋友们不太友好。于是，我参考了这篇文章，使用 LeanCloud 和 Valine 实现了评论功能。同时用LeanCloud也可以很方便地获得浏览量的统计。在测试头像功能的时候，一个人贡献了数十加的浏览量hhh

网络上的资料大多都是基于他人的主题配置评论系统，而我需要在自定义主题中实现。不过步骤大致相同，我成功地在自己的主题中集成了评论系统。现在，你们可以随时在文章下方留下任何的想法和评论啦！

同时用LeanCloud也可以很方便地获得浏览量的统计。

给博客引入订阅功能

在读过这篇文章之后，我便尝试用Inoreader订阅了一些个人博客、周刊等媒介，可能每周会打开Inoreader看一看，也许会幸运地输入一些新的信息，逃离信息茧房。现在我的个人博客也可以通过RSS订阅了。

RSS订阅源来阅读有一定的门槛，你也可以选择收藏我的网站，在浏览器的收藏栏里也能方便地进入网站。
